  • Étape 1 : Ouvrir les options de discussion

    Entrez dans la discussion WhatsApp que vous souhaitez exporter.

    Cliquez sur les trois petits points situés en haut à droite de l'écran pour ouvrir le menu des options.

    Sur Iphone il faut plutot cliquer sur le nom de la personne et aller tout en bas

  • Étape 2 :Accéder à l'option d'exportation

    Dans le menu qui s'affiche, sélectionnez l'option "Plus" (ou "More" si votre WhatsApp est en anglais).

    Choisissez ensuite "Exporter le chat" ou "Export chat".

  • Étape 3 : Choisir le format d'exportation

    Lorsque WhatsApp vous demande si vous souhaitez inclure les médias, sélectionnez "Sans médias" ou "Without media".

    Le téléphone vous demandera alors où vous souhaitez enregistrer le fichier ZIP contenant la discussion. Il est conseillé de l'enregistrer dans le dossier **Téléchargements** de votre explorateur de fichiers

  • Étape 4 : Télécharger le fichier sur le site

    Une fois sur le site web où vous souhaitez traiter la discussion, remplissez le formulaire requis.

    Cliquez sur le bouton pour télécharger le fichier. Cela ouvrira une fenêtre de sélection.

    Dans cette fenêtre, accédez au dossier Téléchargements ou à l'endroit où vous avez stocké le fichier ZIP. Le fichier est généralement nommé "Discussion WhatsApp avec [nom]...zip".

    Sélectionnez le fichier ZIP, puis validez pour finaliser le processus.
